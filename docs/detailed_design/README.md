# CAN Trace Explorer Detailed Design

This directory contains the detailed design derived from the architecture in
[`../architecture`](../architecture) and aligned with the coding and naming
rules in [`../guide_lines.md`](../guide_lines.md).

The detailed design set is organized as:

- `01_design_rules.md`
  Detailed-design level rules derived from the style guide and architecture
- `02_component_detailed_design.md`
  Consolidated detailed design for all software components
- `components/`
  One detailed-design document per software component

No blocking inconsistency has been identified so far between the style guide
and the current architecture proposal. If such a conflict appears during later
implementation, it should be resolved before code is committed.
