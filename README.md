# **CAN Analyzer**

canpp is a lightweight CAN log viewer for exploring and analyzing CAN traffic across formats like BLF, ASC, and others. Developed with C++, it provides powerful search and filtering tools, detailed log inspection, and DBC-based decoding to map frame IDs to message and signal names for a more readable and efficient workflow.
